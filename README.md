# Web DevTools+ app for iOS

A powerful collection of tools for web developers to test and debug their websites on iOS and iPadOS.

## Features

- CSS Quick Tools with overlays to debug overflow issues, grid/flex issues
- Performance Snapshot to test and debug performance issues
- Accessibility Snapshot to test and debug accessibility issues
- SEO Check to audit meta tags, structured data, and other SEO issues
- Storage Snapshot to test and debug storage issues like cookies, local storage, session storage, etc.
- Rendering panel to test and debug rendering issues
- Accessibility panel to test and debug A11y issues
